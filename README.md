# CryptoStatix - Premium Landing Page

I've successfully created the **CryptoStatix** landing page. It is a high-performance, institutional-grade web application with a focus on modern aesthetics and smooth user experience.

## ✨ Key Features

-   **Institutional Design**: A dark-themed, sleek interface with emerald green accents and glassmorphism effects.
-   **Real-time Crypto Ticker**: An animated marquee displaying the latest prices and 24h volatility for top cryptocurrencies.
-   **Interactive Navigation**: Smooth transitions between the Home, Analytics, and Screener sections.
-   **Custom Background**: Integration of the high-resolution trading desk image you provided.
-   **Responsive & SEO Ready**: Optimized for different screen sizes with proper meta tags.

## 🛠️ Technical Stack

-   **Vite**: For lightning-fast development and optimized production builds.
-   **Vanilla JS & CSS**: Maximum performance with zero unnecessary overhead.
-   **Google Fonts**: 'Orbitron' and 'Inter' for that futuristic, data-driven look.

## 📁 Project Structure

-   `index.html`: Main structure with SEO metadata.
-   `src/main.js`: Business logic, ticker initialization, and page transition effects.
